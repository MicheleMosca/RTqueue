Constructor of a FIFOLinkedQueue. dimension, blocked and persistence are unmandatory values. Default value for blocked is true (Push and Pop function will be blocked functions). Default value for dimension is UNLIMITED. Default value for persistence is true (only a pop() function can be remove element inside the queue).
